=======================================================
🐠 어종 판별 웹사이트 (Fish Species Classifier Web App)
=======================================================

## 1. 프로젝트 소개

이 프로젝트는 사용자가 이미지를 업로드(PC)하거나 직접 사진을 촬영(모바일)하여, 미리 학습된 인공지능 모델이 해당 이미지 속의 어종을 판별해 주는 웹사이트입니다.

### 판별 가능한 어종:
* 넙치 (Olive Flounder)
* 볼락 (Korea Rockfish)
* 참돔 (Red Seabream)
* 감성돔 (Black Porgy)
* 돌돔 (Rock Bream)

모델이 어종을 확실하게 판별할 수 없거나, 물고기가 아닌 사진이라면 '확인할 수 없습니다.'라는 메시지를 출력합니다.

## 2. 시작하기 전에 (필수 준비물)

이 웹사이트를 실행하기 위해서는 다음 준비물이 필요합니다.

* Python 3.8 이상: 파이썬이 컴퓨터에 설치되어 있어야 합니다. (개발 환경 설정을 위해 Anaconda 사용을 권장합니다.)
* Git: 소스 코드를 다운로드하고 관리하기 위해 필요합니다. (GitHub에서 클론할 경우)
* PIP: 파이썬 패키지 설치 도구 (일반적으로 Python 설치 시 자동으로 포함됩니다).
* `best.pt` 파일: 어종 판별 모델의 핵심 파일입니다. 이 README.txt 파일과 `app.py` 파일이 있는 동일한 폴더에 `best.pt` 파일이 있어야 합니다.
* 인터넷 연결: 필요한 라이브러리 설치 및 모델 구동을 위해 필요합니다.

## 3. 설치 및 실행 방법

다음 단계를 순서대로 따라주세요.

### 단계 1: 프로젝트 파일 다운로드

1. 명령 프롬프트(Windows) 또는 터미널(macOS/Linux)을 엽니다.
2. 이 프로젝트 파일을 다운로드합니다. (예: 압축 파일 해제 또는 Git 클론)
   * Git 사용 시:
     ```
     git clone [이 프로젝트의 GitHub 저장소 URL]
     cd [클론된 폴더 이름, 예: FishSpeciesClassifierWebApp]
     ```
   * 압축 파일 사용 시:
     다운로드한 압축 파일을 해제한 후, 해당 프로젝트 폴더로 이동합니다.
     (예: `cd C:\Users\사용자명\Desktop\GitTestTeam`)

### 단계 2: 필요한 라이브러리 설치

라이브러리 설치에는 두 가지 방법이 있습니다. 가상 환경 사용을 강력히 권장합니다.

**방법 A: 가상 환경 사용 (권장)**
프로젝트 간의 의존성 충돌을 방지하고 깔끔한 개발 환경을 유지하기 위한 가장 안전하고 권장되는 방법입니다.

1.  **가상 환경 생성:** 프로젝트 폴더 내에서 다음 명령어를 실행합니다.
    ```
    python -m venv venv
    ```
2.  **가상 환경 활성화:**
    * Windows:
        ```
        .\venv\Scripts\activate
        ```
    * macOS / Linux:
        ```
        source venv/bin/activate
        ```
    (성공적으로 활성화되면 명령 프롬프트/터미널 프롬프트 앞에 `(venv)`가 표시됩니다.)
3.  **라이브러리 설치:** 가상 환경이 활성화된 상태에서, 필요한 모든 라이브러리를 설치합니다.
    ```
    pip install Flask Pillow torchvision torch
    ```

**방법 B: 가상 환경 없이 실행**
가상 환경 없이 시스템 전역에 라이브러리를 설치하는 방법입니다. 다른 파이썬 프로젝트와 라이브러리 버전 충돌을 일으킬 수 있으므로 주의가 필요합니다.

1.  (현재 가상 환경이 활성화되어 있다면) 가상 환경을 비활성화합니다:
    ```
    deactivate
    ```
2.  **라이브러리 설치:** 다음 명령어를 입력하여 필요한 라이브러리를 시스템 전역에 설치합니다.
    ```
    pip install Flask Pillow torchvision torch
    ```

*설치 과정은 인터넷 연결 상태에 따라 몇 분 정도 소요될 수 있습니다.*

### 단계 3: 웹사이트 실행

이제 모든 준비가 완료되었습니다. 다음 명령어를 입력하여 웹사이트를 실행합니다.


* 정상적으로 실행되면 터미널에 다음과 유사한 메시지가 표시됩니다:
    ```
    * Running on all addresses (0.0.0.0)
    * Running on [http://127.0.0.1:5000](http://127.0.0.1:5000)
    * Running on http://[당신의 PC 로컬 IP 주소]:5000  (예: [http://192.168.219.65:5000](http://192.168.219.65:5000))
    Press CTRL+C to quit
    ```

### 단계 4: 웹사이트 접속

1. 웹 브라우저를 엽니다 (Chrome, Firefox, Edge, Safari 등).
2. 터미널에 표시된 IP 주소와 포트 번호로 접속합니다.
   * PC에서 접속 시: `http://127.0.0.1:5000`
   * 모바일/다른 기기에서 접속 시: `http://[당신의 PC 로컬 IP 주소]:5000` (예: `http://192.168.219.65:5000`)
   * **중요:** 모바일 기기는 PC와 **동일한 Wi-Fi 네트워크**에 연결되어 있어야 합니다.

## 4. 웹사이트 사용 방법

* PC 환경:
    * 웹사이트 중앙의 점선 영역("여기에 이미지를 드래그 앤 드롭하거나 클릭하여 업로드하세요")에 이미지 파일을 끌어다 놓습니다.
    * 또는 해당 영역을 클릭하여 파일 탐색기에서 이미지를 선택합니다.
* 모바일 환경:
    * 점선 영역을 클릭하여 갤러리에서 이미지를 선택하거나, "📸 사진 찍기" 버튼을 눌러 카메라를 사용하여 사진을 찍고 업로드합니다.
* 결과 확인:
    * 업로드된 이미지는 미리보기로 표시됩니다.
    * 잠시 후 이미지 아래에 어종 판별 결과가 한국어 및 영어 이름으로 표시됩니다.
    * 만약 모델이 어종을 확실하게 판별할 수 없거나, 물고기가 아닌 사진이라면 "🚨 확인할 수 없습니다."라는 메시지가 표시됩니다.

## 5. 문제 해결 (Troubleshooting)

* `FileNotFoundError` (best.pt 파일을 찾을 수 없습니다):
    * `best.pt` 파일이 `app.py` 파일과 같은 폴더에 있는지 확인해주세요.
* `ModuleNotFoundError` (특정 모듈을 찾을 수 없습니다):
    * **단계 3**에서 필요한 라이브러리가 모두 설치되었는지 확인해주세요. (가상 환경 사용 시 활성화되었는지 확인)
* 모바일/다른 기기에서 접속이 안 돼요 (`사이트에 연결할 수 없음`):
    * **가장 흔한 문제:** PC와 모바일 기기가 **동일한 Wi-Fi 네트워크**에 연결되어 있는지 확인합니다.
    * PC의 로컬 IP 주소와 포트 번호(`5000`)를 정확히 입력했는지 확인합니다.
    * PC의 **Windows Defender 방화벽**이 5000번 포트의 인바운드 연결을 차단하고 있을 수 있습니다. 방화벽 예외 규칙을 추가하거나, 테스트 목적으로 잠시 방화벽을 비활성화해 보세요.
    * 네트워크 연결 상태가 불안정할 수 있으니, PC와 모바일의 Wi-Fi를 껐다가 켜거나 공유기를 재부팅해보세요.
* "사진 찍기" 버튼을 눌러도 카메라가 작동하지 않아요 (모바일):
    * 모바일 브라우저에 카메라 접근 권한이 허용되어 있는지 확인합니다. (기기 설정 -> 앱 -> 해당 브라우저 -> 권한)
    * HTTP 환경에서는 일부 브라우저가 카메라 접근을 제한할 수 있습니다. (고급: `ngrok` 등을 통한 HTTPS 터널링 고려)
* 이미지 업로드/촬영 후 "오류 발생" 또는 "판별 중..." 상태로 멈춤:
    * PC의 Flask 서버(CMD 창)에 오류 메시지(Traceback)가 출력되는지 확인합니다. 해당 오류 메시지를 검색하여 해결해야 합니다.
    * 간혹 이미지 파일 형식이나 크기 때문에 문제가 발생할 수 있습니다.

## 6. 종료하기

* 웹사이트 실행을 중지하려면, `app.py`가 실행되고 있는 명령 프롬프트(터미널) 창을 선택한 후 `Ctrl + C`를 누릅니다.
* (가상 환경을 사용했다면) 가상 환경을 비활성화하려면:
    ```
    deactivate
    ```
